# Reaction–Diffusion (RD)

このアプリは、**反応拡散（Reaction–Diffusion）** をリアルタイムに可視化するシミュレータです。  
Turing が着想したパターン生成の枠組みをベースに、濃度の拡散と化学反応（生成/消滅）を数値計算で解き、
**ヒョウ柄・シマ模様・斑点から迷路状の紋様まで**を画面上で生成します。
